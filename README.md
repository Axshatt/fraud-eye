Advanced Fraud Detection and Analysis System with Apple-style UI aesthetics.

## Features
- **Transaction Analysis:** AML heuristics and relationship mapping.
- **Visual Graph:** Interactive graph visualization of transaction networks.
- **Premium UI:** Polished Apple-style whitish aesthetic with smooth animations.
- **Autofill:** One-click sample data injection for quick testing.

## Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Framer Motion, Lucide React.
- **Backend:** Node.js, Express, MongoDB.
- **Utilities:** `asyncHandler`, `ApiError`, `ApiResponse` constructors.

## Getting Started

### Prerequisites
- Node.js installed
- MongoDB URI (added to `.env`)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/CodexAde/fraud-eye.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables in `backend/.env` and `frontend/.env`.

### Running the App
```bash
npm run dev
```
